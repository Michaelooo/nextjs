# 一个简单的react应用例子，使用 nextjs , parceljs, yarn ……

一个简单的react应用例子，使用 nextjs , parceljs, yarn ……，会使用最新最热的几门东东，仅供学习使用

## 使用：

```
<!-- 克隆本地 -->
git clone git@github.com:Michaelooo/nextjs.git

<!-- 安装依赖 -->
yarn install 

<!-- 开启服务 -->
yarn run dev
```

## 问题：
